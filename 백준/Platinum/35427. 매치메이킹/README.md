### 문제 설명

<p>PVP 게임을 만들고 있는 도윤이는 매치메이킹 시스템을 구현하려고 합니다. 각 사람은 저마다 레이팅을 가지고 있고, 게임은 1:1 방식으로 이루어져 현재 매치메이킹 시스템에 있는 사람 중 레이팅 차이가 최소인 두 사람을 매칭하는 방식으로 이루어집니다. 도윤이를 대신해서 아래 쿼리 $Q$개를 처리하는 프로그램을 작성해봅시다.</p>

<ul>
<li>$1$ $x$ $r$: 번호가 $x$, 레이팅이 $r$인 사람을 매치메이킹 시스템에 추가합니다. 이때 서로 다른 $1$번 쿼리에서 $x$의 값은 항상 서로 다르며, 번호와 레이팅은 모두 양의 정수입니다.</li>
<li>$2$: 레이팅 차가 최소인 정확히 한 쌍의 매칭을 만듭니다. 이런 쌍이 여럿이면 그중 아무 쌍이나 매칭해도 됩니다. 이 쿼리가 들어오면 매칭된 두 사람의 번호를 공백으로 구분해 출력하고, 두 사람을 매치메이킹 시스템에서 제거해야 합니다. 이때 매칭된 두 사람의 번호를 출력하는 순서는 무관합니다.</li>
</ul>

<p>매치메이킹 시스템에 $2$명 이상이 남아 있을 때만 $2$번 쿼리가 주어지고, 전체 쿼리 중 적어도 하나 이상 $2$번 쿼리가 주어집니다.</p>

### 입력 

 <p>첫째 줄에 전체 쿼리의 수 $Q$가 주어집니다.</p>

<p>둘째 줄부터 $Q$개 줄에 걸쳐 쿼리가 주어집니다.</p>

### 출력 

 <p>$2$번 쿼리가 주어질 때마다 각 줄에 매칭된 두 사람의 번호를 공백으로 구분하여 출력합니다.</p>
